What is HTML?

HTML stands for Hyper Text Markup Language. It describes the structure of web pages.

HTML in the above sentence is italicised because it is within the dfn tag which is used to refer to the first instance of a definition.

Head and Body

  1. Head : This section contains the meta data of a web page. This is information about the page, not displayed on the page. The title can be visible on the tab.
  2. Body: This is the main structure of the page that is visible to the user.

Elements and Tags

HTML comprises of a series of elements that tell the browser how to display the content so that we can format the page just like we do a word document.

Tags
Tags are used to mark up content on a webpage. They define the structure and meaning of the content they enclose. For example: <p>

The p tag above is diplayed within the code tag by escaping characters in HTML.Note that the link is an a tag with attribute href pointing to the url.

Elements
The opening tag, closing tag and the content within them are together called HTML elements.
For example, <p> This list is created using definition list, dl with dt for definition term and dd for definition </p>
Attributes
Attributes are used to modify the behavior or appearance of an element or to provide additional information that cannot be conveyed through the element's content alone.
  • href in a tag
  • src in img tag
  • alt in img tag
  • style attribute
  • lang attribute

Structural vs Semantic Markup

Structural markup is used to define the hierarchical structure of a document, but it does not provide any information about the meaning or purpose of the content within those structures.
Semantic markup, on the other hand, is the use of HTML tags that convey meaning and context to the content within those structures.

Images

    Some points to consider
    • select the right format -
    • png for flat color(large area of image filled with exacltly the same color) image like logos and diagrams
    • jpeg for images with multiple colors. Note that images with sky, snow, etc. are actually made of different shades although they look like 1 -2 colors. So use jpeg for these.
  • Select image of right size, dimensions in pixels and not in cm or in.
  • Provide an accurate alt attribute description. If image is only for decorative purposes, still provide alt attr with empty quotes.

Form

Forms are important HTML elements that are used to collect information from users. An HTML form constitutes various form controls to gather different types of data from the user.

The name of the form control and the value entered by the visitor are sent to the server.
